📌 Projeto React - Aplicação Web com Login Google
📖 Descrição do Projeto

Este projeto consiste no desenvolvimento de uma aplicação web utilizando React, com foco em autenticação via Google e cadastro de usuários.

A aplicação permite que o usuário realize login com sua conta Google e, a partir disso, utilize suas informações para preencher automaticamente um formulário de cadastro, gerando um objeto JSON com os dados inseridos.

O sistema foi desenvolvido com base em boas práticas de Front-end e organização de componentes.

🚀 Funcionalidades
🏠 Tela Inicial (Home)
Interface simples e intuitiva;
Login com conta Google;
Armazenamento das informações do usuário:
  Nome
  E-mail
  Foto
Navegação para outras telas.

👩‍💻 Tela de Apresentação da Dupla
Exibição dos integrantes do projeto;
Informações sobre cada desenvolvedor:
  Formação
  Habilidades
  Interesses
Layout organizado e visual agradável.

📝 Tela de Cadastro de Usuário
Formulário com os campos:
  Nome
  E-mail
  Telefone (opcional)
  Outros dados relevantes
Preenchimento automático com dados do login Google;
Geração de objeto JSON com os dados cadastrados;
Exibição do JSON na tela ou console.

⚙️ Tecnologias Utilizadas
⚛️ React
📦 Vite (ou Create React App, se aplicável)
🔀 React Router
🔐 Google Authentication (OAuth / Firebase / Google Cloud)
🎨 CSS Modules
💻 JavaScript / TypeScript

🧠 Conceitos Aplicados:
Componentes funcionais

Gerenciamento de estado
Navegação entre páginas
Manipulação de formulários

🔐 Configuração do Login Google
Para funcionamento correto do login Google:

Criar projeto no Google Cloud Console
Configurar OAuth Client ID
Adicionar URLs autorizadas (incluindo Netlify)
